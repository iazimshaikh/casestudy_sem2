Becoming a Hacker

Hello friends !

So first question will be coming in your mind, I want to become a hacker but I am confused from where to start? Don't worry let's understand..

Who is called a Hacker?

So basically a hacker is a person who can find a vulnerability or bug in the system or network okay. Now keep this definition in your mind for your basic understanding. And secondly hacker is a person "who can access any system without person's or victim's knowledge" okay.. But just think when you will be able to do this job? Let's understand it through an example. Suppose, you are going to hack someone's windows machine (computer).

"You will be able to do this work only when you know everything about the Windows operating system."

Do you get my point? If yes than tell me in the comment section. So the first thing you should learn is windows operating system (just because windows is a most popular and commonly used operating system).

"But which things you should learn in windows?" This is the very biggest question ????

Okay I give you an example of mine.

"First of all, What I had done to become a hacker?"

  1. Study about 'WINDOWS OPERATING SYSTEM', What is Operating System?, Difference between all Operating Systems like Mac, Kali Linux, Windows, etc.
  2. Study about various important files of Windows operating systems as an example - "SAM" file which is 'Security Account Manager', 'Host' file, etc which are the important files of Windows operating system.
  3. Study about how Windows operating system works.
  4. Learn about computer hardware.
  5. Learn about computer networking.
  6. Learn about computer programming languages such as Python, Java, Php, C.
  7. Learn about Unix-based operating systems such as Kali Linux, Parrot Operating System, Black Arch, and Backtrack.
  8. Learn about social engineering (most important).

These are the important topics which you must know about... in the field of Ethical hacking.
