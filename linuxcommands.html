Welcome guys to my blog. In this blog I will show you commands which are the rarely known commands. So lets move on to the topic. 

1. file ( name of file) :- Shows the type of   file. 
     Ex. file password  :- ASCII TEXT

2. wc ( name of file) :- Shows the lines, words and letters of the input provided to that file. 
      Ex. wc linux :- 11 67 1234 /linux
     This file has  11 lines, 67 words and  letters . 

3. w :- w command shows which users are logged in to the system. 

4. history :- History command shows the commands you have used or entered in your terminal. 

5. passwd :- To change your password. 

6. pwd :- shows the current directory. 

7. tty :- Displays current terminal. 

8. sudo :- This command will give you  permission. 
 Ex. sudo apt-get update 
( sometimes when you type only apt-get update it shows E :unable to locate directory /var/lib/apt/list. .) 

9. python  -m  SimpleHTTPServer :- This command is used to share files and folders through port 8000.
( now you have to just type your lhost ip and :8000 in your browser.. And you will get it..) 

10.  curl ifconfig.me :- This command shows external ip address of the machine. 

11. Su :- Gives you root permission. 

12. sudo useradd (username) :- Adds a new user. 

13. sudo userdel (username) :- Delete the user. 

14. mkfs :- It formats the partition. 

15. fdisk :- It manages all the partition on the hard disk. 

16. fsck :- It checks for error in partition. 

17. dd :- It makes copy of input file to an output  file. It clones a disk. 

18. poweroff :- Poweroff the machine.
