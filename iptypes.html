                            PRIVATE IP 

1. Private ip is used to communicate within the same network means on LAN .

2. It works only on LAN ( Local Area Network )

3. This ip address is given at free of cost

4. You can get your private ip by just typing ' ipconfig / ifconfig in terminal.

5. Ranges of private ip is shown as below :- 

            > 10.0.0.0  to  10.255.255.255

            > 172.16.0.0  to  172.31.255.255

            > 192.168.0.0  to  192.168.255.255

     Example :- 192.168.1.0     

                            PUBLIC IP 

1. Public ip is used to communicate outside the same network.

2. Public ip is used for internet service .

3. Public ip is not free , charge is taken for this.

4. Public ip is provided by ISP.

5. Ranges of public ip is as shown below :-

                > Class A :- 1 to 126

                > Class B :- 128 to 191 

                > Class C :- 191 to 223 

   Note :- There are 3 classes of public ip as 

                 shown above Class A, B and C.

                     Example :- 17.5.6.7
