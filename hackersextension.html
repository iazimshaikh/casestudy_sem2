Extensions Used By The Hackers

Extensions mostly used by the hackers

Didn't you feel that you should also use those extensions used by hackers! So, today I will show you some extensions used by the hackers, which you can also use for your dream.


1. HACKBAR

Hackbar is an extension specially made for pentesters for web pentesting. Hackbar allows you to test web security by attacking on the web with the help of SQL Injection, XSS, etc. And it has GUI and CLI interface/user-friendly interface which helps you more comfortably to use.

Links to add this extension:

  • Hackbar Chrome
  • Hackbar Firefox

2. WAPPALYZER

Wappalyzer is an extension that helps us to collect information related to a web application which will be useful in the task of performing pentesting.

Links to add this extension:

  • Wappalyzer Chrome
  • Wappalyzer Firefox

3. NETCRAFT

Netcraft is an extension mainly used by people who have fear of being hacked by the hackers. It scans for vulnerabilities and provides internet security settings. When you visit any website you can see if the website is secure or not and all that type of content with the help of this extension. I will prefer you to use this extension.

Links to add this extension:

  • Netcraft Chrome
  • Netcraft Firefox

4. D3CODER

D3coder is an extension which helps to decide various hash format codes easily and it is very necessary in the field of hacking that you should know how to encode and decode code. NOTE :- Same extension not available for firefox another similar link is given

  • D3coder Chrome
  • D3coder Firefox

5. USER AGENT SWITCHER

User agent switcher is a extension mostly used by cyber security researcher and hackers. It helps to make server bullshit in impersonating ( retrieving itself as other user while hiding own identity ) it's browser details and also OS details.

  • UAS Chrome
  • UAS Firefox
